📊 Projeto 3— Rede Social com Neo4j (Graph Database)

Este projeto consiste na modelagem de uma rede social utilizando Neo4j, explorando o poder dos bancos de dados em grafos para representar conexões e engajamento entre usuários.

O objetivo principal é simular como pessoas interagem dentro de uma plataforma social, permitindo analisar relacionamentos, comunidades e níveis de engajamento de forma natural através de grafos.

🚀 Conceitos aplicados

Modelagem de dados orientada a grafos

Relacionamentos complexos entre entidades

Análise de conexões sociais

Estruturação de dados para engajamento digital

Linguagem Cypher para consultas e manipulação de dados

🧩 Entidades do sistema

👤 Usuários — pessoas da rede social

📝 Posts — conteúdos publicados

🛍 Produtos — itens promovidos/interagidos

👥 Grupos — espaços de interesse comum

🌐 Comunidades — agrupamentos maiores de usuários

🔗 Relacionamentos modelados

Usuários seguem outros usuários

Usuários criam posts

Usuários participam de grupos e comunidades

Interações com produtos

Engajamentos (curtir, comentar, participar)

🎯 Objetivo do projeto

Demonstrar na prática como bancos de dados em grafos são ideais para:

Redes sociais

Sistemas de recomendação

Análise de influência

Detecção de comunidades

Análise de engajamento entre usuários

🛠 Tecnologias utilizadas

Neo4j

Cypher Query Language

APOC Procedures
